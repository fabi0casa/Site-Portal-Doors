Bem vindo

O sistema operacional Doors é um projeto desenvolvido voluntariamente por nossa equipe, com o objetivo de ser um sistema de código aberto, amigável, privado e de fácil acesso aos usuários comuns de PC.
É inegável dizer que o sistema operacional é uma parte chave na hora de montar e/ou escolher um computador, o mais comum de todos, "Windows" tem sim as suas vantagens, entretanto no quesito de performance, customização e liberdade do usuário sobre o sistema, o Windows não faz o melhor papel. Neste contexto que se insere o novo sistema operacional Doors que visa principalmente melhorar os quesitos em que o Windows peca, mas ainda mantendo uma interface amigável e comum aos alhos de alguem que sempre usou o outro sistema, sendo uma excelente alternativa de migração de SO, além de ser gratuíta e descentralizada.
Ao lado é possível ver uma print do sistema operacional em funcionamento, como é perceptível o usuário optou por uma estética semelhante ao clássico Windows XP, mas isso não significa que todos são assim, muito pelo contrário, quem decide a aparência de seu computador é o usuário que facilmente consegue mudar o tema, os botões e o que mais for desejado.
Por este ser um sistema de código-fonte aberto, qualquer um pode modificar o sistema do jeito que bem entender, gerando diversas versões do mesmo sistema assim como funciona o Linux hoje em dia, com o detalhe que o Doors é totalmente compatível com aplicativos desenvolvidos originalmente para Windows.
